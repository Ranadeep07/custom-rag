# Gemini RAG FastAPI

This project is a **Retrieval-Augmented Generation (RAG)** system using **Google Gemini API**.  
It exposes a **FastAPI** endpoint that takes a query, searches your documents, and returns a generated answer.

---

## Features

- Simple RAG pipeline using **chunks + embeddings + vector store (FAISS)**  
- Uses **Google Gemini API** for embeddings and content generation  
- FastAPI endpoint `/ask` for querying  
- Ready for deployment on **Render, Railway, Fly.io**, etc.
